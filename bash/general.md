# List of general commands

| Description | Syntax |
| ----------- | ----------- |
| Show who is logged into a server | `w` |
| Show bash history | `history` |
| Run the last command as sudo | `sudo !!` |
| Run the history id (10) command as sudo | `sudo !10` |
| Show disk free | `df -h` |
| Show disk mounts | `mount` |
